## How to run Step 1 

1. Run "cd step1"
2. Run "g++ -std=c++11 *.cpp"
3. Run "./a.out OutputAndTestCases/TestCases10_08_22/[Test]" with [Test] being the test case. 
   Ex. To run 3Add, input the following: "./a.out OutputAndTestCases/TestCases10_08_22/3Add"
4. The output will display on the console and in the "result.txt" file

## How to run Step 2

1. Run "cd step2"
2. Run "g++ -std=c++11 *.cpp"
3. Run "./a.out OutputAndTestCases/ParserOutput10_08_22/[Test]" with [Test] being the test case. 
   Ex. To run 3Add.pout, input the following: "./a.out OutputAndTestCases/TestCases10_08_22/3Add.pout"
4. The output will display on the console
